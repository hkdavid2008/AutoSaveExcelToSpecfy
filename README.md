# AutoSaveExcelToSpecfy
thunderbrid Extension
项目招标:
       开发thunderbird的插件，自动分析thunderbird邮件客户端里面最近天数的邮件,如果附件是EXCEL文件和发信地址在指定列表里面，则自动保存EXCEL附件到指定的目录，并重新命名。
技术分析 
1.要求使用WebExtension API技术。 
2.天数可以设置。
3.指定列表为Txt文件，方便编辑。
4.文件保存规范：域名#+原文件名。
